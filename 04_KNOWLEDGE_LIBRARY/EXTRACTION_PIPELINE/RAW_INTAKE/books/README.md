# books/

Drop book content, chapters, and long-form intellectual property here.

**What goes here:** Book chapters, full manuscripts, long-form writing drafts, content for the 4 planned books (property professionals, enterprise OS, fitness, self-development).

**Format:** Markdown (.md), PDF, Word (.docx), plain text (.txt).

**What happens next:**
1. `intake_processor.py` routes to the relevant pillar or project folder
2. `v7_extract.py book <file>` extracts: proofs, arguments, facts, hooks, rhetorical devices, objections/myths, beliefs to challenge, solutions, quotable lines
3. Extraction output saved alongside AND in `EXTRACTIONS/by_mode/book/`

**Naming:** `YYYY-MM-DD_book_title_chapter_NN.md` (e.g. `2026-02-20_property_book_ch03_market_analysis.md`)
