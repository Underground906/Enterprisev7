## KEY FRAMEWORKS EXTRACTED

### 1. The 5A + 5 Dynamic Layers System

```
FOUNDATION (5A):
1. Alignment    - Vision, direction, coherence
2. Awareness    - Risks, gaps, constraints
3. Accountabilities - Outcome ownership (90-day)
4. Activities   - Weekly execution
5. Assets       - Build/buy to compound

DYNAMIC LAYERS:
6. State        - Current reality (AI-refreshed)
7. Decisions    - Course changes logged
8. Learning     - Pattern extraction
9. Iterations   - Version control
10. AI Dialogue - System challenges/alerts
```

### 2. Goal Intake Pipeline

```
CAPTURE → PARSE → GRADE → PLACE → SCHEDULE → TRACK → REVIEW
   ↓        ↓       ↓       ↓        ↓         ↓       ↓
 Raw     Extract  Score   Route    Time      Log    Adjust
 dump    fields   1-10    domain   block     state   course
```

### 3. Goal Classification

```
TYPE: vision | outcome | project | task | problem | awareness
DOMAIN: personal | business | platform | founder | enterprise
SPECIFICITY: vague | directional | defined | detailed
```

### 4. 90-Day Accountability Cycle

- 3-6 critical outcomes per cycle
- ONE clear owner per outcome
- Time-boxed, reviewable, replaceable
- Prevents "identity lock-in"
