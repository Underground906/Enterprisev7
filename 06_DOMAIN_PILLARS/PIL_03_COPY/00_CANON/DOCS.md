# SECTION 2: CONTEXT DOC (For AI/Agents)

## Master Copywriter Formulas

### Headlines
- **SCHWARTZ:** Problem + Amplification + Solution
- **CAPLES:** Specific Benefit + Timeframe
- **KENNEDY:** Exclusivity + Scarcity
- **HALBERT:** Curiosity Gap
- **OGILVY:** News + Benefit

### Offers
```
SCHWARTZ: [Product] + [Bonuses] + [Incentive] = [Value] for [Price]
KENNEDY: [Solution] + [Guarantee] + [Risk Reversal]
BROWN: [Basic] → [Mid] → [Premium] tiers
```

### CTAs
```
URGENCY: "Only [N] spots. [Action] before [Deadline]"
EASY: "To [result], simply [action]"
LOSS: "Leave without [action] = miss [benefit]"
```

## Copy Block Functions

### Attention Blocks
- Problem-Focused Headlines → Pain activation
- Benefit-Driven Headlines → Desire creation
- Curiosity Headlines → Information gap
- Urgency Headlines → Loss aversion

### Trust Blocks
- Testimonials → Social proof
- Case Studies → Transformation evidence
- Authority Markers → Credentials, logos
- Guarantees → Risk reversal

### Conversion Blocks
- Primary CTA → Main action
- Price Reveal → Value justification
- Scarcity → Countdown, limits
- Bonuses → Added value
