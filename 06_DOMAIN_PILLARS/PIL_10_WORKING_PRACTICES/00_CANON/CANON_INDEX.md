### For Daily Work:
1. Read **DAILY_OPERATING_SYSTEM_HUMAN.md** once to understand the system
2. Use the Quick Reference Card daily
3. Follow the 3 rituals: Morning Start, Work Blocks, End of Day

### For Claude Code Sessions:
1. Open **CLAUDE_CODE_SESSION_RUNNER.md**
2. Copy the instruction set

4. Let Claude run you through startup

2. Use wireframes as design reference
