# NAMING CONVENTIONS

## Folder Naming

### Top Level (8 components)
- Format: `NN_NAME` (e.g., `00_SYSTEM_ROOT`, `06_DOMAIN_PILLARS`)
- Always uppercase with underscores

### Pillars
- Format: `PIL_NN_NAME` (e.g., `PIL_03_COPY`, `PIL_19_PROPERTY`)
- Sequential numbering 01-23

### Projects
- Format: `PRJ_Name` (e.g., `PRJ_Property_Connect_London`)
- Title case with underscores

### Goals
- Format: `GOAL_Name` (e.g., `GOAL_AI_Chatbot_Revenue`)
- Title case with underscores

## File Naming

### Canon/System Files
- UPPERCASE with underscores
- Example: `MASTER_CONTEXT.md`, `ROUTING_RULES.md`

### Working Files
- lowercase with underscores
- Example: `session_log.md`, `decisions.md`

### Date-based Files
- Format: `YYYY-MM-DD.md` or `YYYY-MM-DD_description.md`

## Archive Naming
- Always `90_ARCHIVE/` within any folder that needs archiving
- Archived items retain original names
