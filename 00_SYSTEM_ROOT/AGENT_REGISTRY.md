# AGENT REGISTRY

## Purpose
Central registry of all defined AI agents and their capabilities.

## Core Agents

### Research_Agent
- **Location:** 02_COMMAND_DECK/AGENT_WORKSPACE/Research_Agent/
- **Purpose:** Information gathering, market research, competitive analysis
- **Inputs:** Research briefs, questions
- **Outputs:** Research reports, data summaries

### Copy_Agent
- **Location:** 02_COMMAND_DECK/AGENT_WORKSPACE/Copy_Agent/
- **Purpose:** Writing copy across all formats
- **Inputs:** Briefs, brand context, audience data
- **Outputs:** Headlines, body copy, CTAs

### Code_Agent
- **Location:** 02_COMMAND_DECK/AGENT_WORKSPACE/Code_Agent/
- **Purpose:** Development tasks, scripting, automation
- **Inputs:** Requirements, specs
- **Outputs:** Code, scripts, technical docs

### Design_Agent
- **Location:** 02_COMMAND_DECK/AGENT_WORKSPACE/Design_Agent/
- **Purpose:** UI/UX guidance, design system work
- **Inputs:** Design briefs, brand guidelines
- **Outputs:** Design specs, component definitions

### Validator_Agent
- **Location:** 02_COMMAND_DECK/AGENT_WORKSPACE/Validator_Agent/
- **Purpose:** Quality control, validation, review
- **Inputs:** Outputs from other agents
- **Outputs:** Validation reports, corrections

### Orchestrator_Agent
- **Location:** 02_COMMAND_DECK/AGENT_WORKSPACE/Orchestrator_Agent/
- **Purpose:** Coordinate multi-agent workflows
- **Inputs:** Complex tasks requiring multiple agents
- **Outputs:** Coordinated execution plans, task delegation
