## Visual Treatment Selection

```
BRAND LEVEL:
├── Logo → Primary/Secondary/Mono/Animated
├── Colors → Primary/Secondary/Semantic/Dark
└── Typography → Headline/Body/UI/Display

PAGE LEVEL:
├── Hero → Style (20 options) + Treatment
├── Background → Pattern/Texture (18 options)
└── Sections → Illustration style

COMPONENT LEVEL:
├── Icons → Style (15) + Category (20) + Size
├── Buttons → Visual treatment
├── Cards → Photo treatment + shadow
└── Data → Chart type + style

ASSET LEVEL:
├── Photography → Category + Style + Treatment
├── Mockups → Device + Scene + Angle
├── Product → Shot style + Background
└── Animation → Lottie type + Trigger
```

## Effect Stack

```
BASE IMAGE
  ↓ Color Treatment (duotone, grading, etc.)
  ↓ Overlay (light leaks, grain, etc.)
  ↓ Shadow/Depth (drop, ambient)
  ↓ Final adjustments (sharpen, vignette)
OUTPUT
```
