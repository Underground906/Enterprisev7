# SESSION END PROTOCOL

## On Session Close

### 1. Capture Outputs
- What was created this session?
- Where should outputs be saved?
- Route to correct pillar/project folders

### 2. Update State
- What changed?
- What decisions were made?
- Update STATE_SNAPSHOT if significant progress

### 3. Log Session
- Save session summary to 02_COMMAND_DECK/ACTIVE_SESSIONS/
- Note key decisions in decisions.md
- Flag any items for follow-up

### 4. Identify Next Actions
- What needs to happen next?
- Add to TASK_QUEUE if appropriate
- Update goal progress if goal-related

## Cleanup Checklist
- [ ] All outputs saved to correct locations
- [ ] Session logged
- [ ] State updated if needed
- [ ] Next actions identified
- [ ] No loose threads left untracked
