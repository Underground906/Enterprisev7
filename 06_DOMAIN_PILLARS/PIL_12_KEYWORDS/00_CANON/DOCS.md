# SECTION 6: KEY FRAMEWORKS SUMMARY

## Universal Keyword Formula

```
[Modifier] + [Product/Service] + [Intent] + [Location] + [Qualifier]

Examples:
- best + mortgage broker + for first-time buyers + in London + 2026
- affordable + flat + to rent + near me + pet-friendly
- how to + sell property + quickly + in London + without agent
```

## Keyword Types

| Type | Example | Volume |
|------|---------|--------|
| Head | "london property" | High |
| Body | "buy flat london" | Medium |
| Long-tail | "buy 2 bed flat zone 2 under 500k" | Low |
| Question | "how much deposit first time buyer london" | Medium |

## Research Workflow

```
1. DEFINE   → Pillar, category, segment
2. EXTRACT  → Primary (5-10), LSI (10-20), Long-tail (10-20), Questions (10-20)
3. PERMUTE  → Apply 76 patterns + modifiers → 50-100 per keyword
4. STORE    → Database with volume/difficulty
5. DEPLOY   → SEO targets, content calendar, ad campaigns
```
