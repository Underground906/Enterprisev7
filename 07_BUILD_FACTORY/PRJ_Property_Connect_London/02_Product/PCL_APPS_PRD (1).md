# PCL APPS PRD
## B2B AI App Suites for London Property Professionals

---

## OVERVIEW

PCL Apps is a productized SaaS business offering AI-powered tools to London property professionals. Instead of custom builds, these are pre-built apps deployed with client branding in 2-16 hours.

**Model:** Build once → Deploy many → Charge monthly

**Target Market:** 
- 5,000+ estate agencies in London
- 15,000+ branches
- Mortgage brokers, letting agents, investors, developers, contractors

**Why It Works:**
- One deal commission (£8,250 avg) pays for a year of most tools
- Professionals are underserved by tech
- Early AI adopters will dominate
- Recurring revenue compounds

## DEPLOYMENT MODEL

### Build Time by Complexity

| Tier | Build Time | Examples |
|------|------------|----------|
| Simple | 2-4 hours | Calculators, FAQ bots, basic chatbots |
| Medium | 4-8 hours | Lead qualifiers, report generators |
| Complex | 1-2 days | Full dashboards, voice AI, portals |

### Deployment Process

1. **Sale** (Day 0)
   - Demo → Contract → Payment

2. **Onboarding** (Day 1-2)
   - Branding assets collected
   - Data integrations set up
   - Customization requirements

3. **Deployment** (Day 2-3)
   - App configured
   - Client branding applied
   - Testing

4. **Launch** (Day 3-5)
   - Embed code provided
   - Training call
   - Go live

5. **Support** (Ongoing)
   - Slack/email support
   - Monthly check-in
   - Usage reporting

### Tech Stack

- **Frontend:** React, Next.js
- **AI:** Claude API, OpenAI
- **Database:** Supabase, PostgreSQL
- **Hosting:** Vercel, Railway
- **Integrations:** Zapier, Make, direct APIs
- **Voice:** Vapi, Bland.ai
- **Payments:** Stripe
