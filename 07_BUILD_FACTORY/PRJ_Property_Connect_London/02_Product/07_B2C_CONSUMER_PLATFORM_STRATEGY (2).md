```
         │               │               │
         ▼               ▼               ▼
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│  B2B API    │  │  B2C Web    │  │  B2C App    │

│   Label)    │  │  Connect)   │  │             │
└─────────────┘  └─────────────┘  └─────────────┘
      │               │               │
      ▼               ▼               ▼
   Agent's         Your            Your
   Website        Platform          App
```

### Multi-Tenant + Platform Architecture

```javascript
// Same calculation, different deployment
async function analyzeProperty(propertyId, context) {
  const property = await getPropertyData(propertyId);
  const analysis = await runAnalysis(property);
  
  if (context.type === 'b2b') {
    // White-label output with client branding
    return formatForB2B(analysis, context.tenant);
  } else if (context.type === 'b2c') {
    // Platform output with PCL branding
    return formatForB2C(analysis);
  }
}
```
