## PRODUCT DEFINITION

**One line**: AI-powered conversion intelligence platform that identifies, engages, captures, routes, and follows up with every website visitor — so businesses stop losing 95% of their traffic.

**Target clients**: Estate agents, home goods stores, premium ecommerce, professional services.

**Positioning**: Not a chatbot. Not a CRM. Not analytics. ALL of it in one pixel.

**Revenue model**: £5,000/month, limited to [X] clients, webinar launch, high-touch onboarding.

## PART 5: TECH REQUIREMENTS

### APIs Needed (accounts to set up)
- [ ] IPinfo.io — IP-to-company ($99/mo)
- [ ] Apollo.io — contact enrichment (free tier to start)
- [ ] Claude API or OpenAI — AI concierge LLM
- [ ] Resend or SendGrid — transactional email
- [ ] Twilio — SMS notifications
- [ ] Slack API — webhook notifications
- [ ] Cal.com or Calendly — demo booking

### Infrastructure
- [ ] VPS/Cloud for core platform (Railway, Fly.io, or DigitalOcean)
- [ ] PostgreSQL (managed or self-hosted)
- [ ] Redis (session cache + real-time)
- [ ] ChromaDB or Pinecone (vector store for RAG)
- [ ] Domain + DNS
- [ ] SSL certificate
- [ ] Docker registry (for client containers)

### Per-Client Deployment
- Docker Compose with: API + Worker + Redis + ChromaDB
- Client connects their own PostgreSQL or uses managed
- Environment variables for: API keys, brand config, LLM provider
- Nginx reverse proxy with client's subdomain
