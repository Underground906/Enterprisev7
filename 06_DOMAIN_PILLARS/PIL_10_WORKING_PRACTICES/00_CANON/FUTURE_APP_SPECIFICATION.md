## TECHNICAL ARCHITECTURE

### Backend
- Database: PostgreSQL (users, sessions, outputs, state)
- Auth: JWT with refresh tokens
- API: REST or GraphQL
- File storage: S3-compatible for outputs
- Real-time: WebSockets for team presence

### Frontend
- Framework: React or Next.js
- State: Redux or Zustand
- UI: Tailwind + shadcn/ui
- Editor: Monaco (code) or TipTap (docs)
